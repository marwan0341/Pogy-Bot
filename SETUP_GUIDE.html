Pogy Bot Deployment Guide

Simple Setup Guide

Yeh guide easy Hinglish me hai. Isme dashboard ko Vercel par aur bot ko Pterodactyl par kaise chalana hai woh step by step samjhaya gaya hai.

Dashboard

Vercel me kya jayega

  • Sirf dashboard folder host karna hai.
  • Vercel project root dashboard hona chahiye.
  • Login, dashboard UI, aur API routes yahin chalenge.
Bot

Pterodactyl me kya jayega

  • Poora bot project upload karna hai.
  • Main start file index.js hi rahegi.
  • Bot API same process ke andar chalegi.
Step 1

Vercel Setup

  1. GitHub repo ko Vercel se import karo.
  2. Project root directory dashboard set karo.
  3. Build command npm run build rakho.
  4. Output directory dist rakho.

Vercel env values

DISCORD_CLIENT_ID=1489584892852441171 DISCORD_CLIENT_SECRET=YOUR_CLIENT_SECRET DISCORD_BOT_TOKEN=YOUR_BOT_TOKEN MONGODB_URI=YOUR_MONGODB_URI NEXTAUTH_SECRET=CHANGE_THIS_TO_LONG_RANDOM_SECRET DASHBOARD_PUBLIC_URL=https://pogy-showcase.vercel.app BOT_API_BASE_URL=http://n3.nexcloud.in:2053 DASHBOARD_BOT_STATUS_URL=http://n3.nexcloud.in:2053 SUPPORT_URL=https://discord.gg/codexdev GUIDE_URL=https://codexdevs.in
Step 2

Discord Developer Portal Setup

  1. Discord Developer Portal kholo.
  2. Apni application select karo.
  3. OAuth2 redirect URL me yeh add karo:
https://pogy-showcase.vercel.app/api/auth/callback/discord

Agar yeh redirect URL sahi nahi hoga, login white page ya fail ho jayega.

Step 3

Pterodactyl Bot Setup

  1. Bot files Pterodactyl server me upload karo.
  2. Startup command ya install command me dependencies install karvao.
  3. Bot env set karo.
  4. Bot run karo.

Bot env values

DISCORD_TOKEN=YOUR_BOT_TOKEN DISCORD_BOT_TOKEN=YOUR_BOT_TOKEN DISCORD_CLIENT_ID=1489584892852441171 DISCORD_CLIENT_SECRET=YOUR_CLIENT_SECRET BOT_PREFIX=? BOT_OWNERS=1434858158618054756,1252101579855630346 MONGODB_URI=YOUR_MONGODB_URI LAVALINK_SEARCH_ENGINE=youtube LAVALINK_RESUME=true LAVALINK_RESUME_TIMEOUT=60 LAVALINK_NODE_NAME=main LAVALINK_NODE_URL=lava-v4.millohost.my.id:443 LAVALINK_NODE_AUTH=YOUR_LAVALINK_AUTH LAVALINK_NODE_SECURE=true WEBHOOK_URL=YOUR_WEBHOOK_URL SUPPORT_URL=https://discord.gg/codexdev GUIDE_URL=https://codexdevs.in TOTAL_SHARDS=auto SHARDS_PER_CLUSTER=4 TOTAL_CLUSTERS=auto STATUS_SERVER_PORT=2053 STATUS_SERVER_HOST=0.0.0.0

Run command

npm install && npm run start:bot
Step 4

Bot API test kaise karna hai

Pehle yeh 3 URLs browser me check karo:

http://n3.nexcloud.in:2053/ http://n3.nexcloud.in:2053/stats http://n3.nexcloud.in:2053/guilds
  • Agar yeh open ho rahe hain to bot API live hai.
  • Agar yeh open nahi ho rahe to dashboard bot se connect nahi karega.
Important

Simple samajh lo

  • Vercel sirf dashboard dikhayega aur login handle karega.
  • Pterodactyl bot ko chalata rahega.
  • Dashboard bot se http://n3.nexcloud.in:2053 ke through baat karega.
  • Bot API public hona zaroori hai.
Warning

Public karne se pehle

  • Bot token rotate karo.
  • Client secret rotate karo.
  • MongoDB password rotate karo.
  • Webhook URL rotate karo.

Purane exposed secrets ke saath public GitHub push mat karo.
